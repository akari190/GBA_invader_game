# GBA Invader Game

## 概要

このリポジトリは、Game Boy Advance (GBA) 向けに制作したインベーダー風シューティングゲームです。
ソースは `インベーダーゲーム/game コメントあり.c` にあり、GBA の VRAM とキー入力を直接操作して描画と制御を行います。

## 収録ファイル

- `インベーダーゲーム/game コメントあり.c`
  - コメント付きの C ソースコード
- `インベーダーゲーム/game.s`
  - コンパイル時に生成されたアセンブリ出力
- `インベーダーゲーム/game.o`
  - オブジェクトファイル
- `インベーダーゲーム/game.bin`
  - GBA 実行バイナリ
- `インベーダーゲーム/game.out`
  - リンカ出力ファイル

## ゲーム内容

- プレイヤーは画面下部を左右に移動し、敵を弾で撃ちます。
- 敵は 2 行 8 列で配置され、移動しながらプレイヤーに向かって弾を撃ってきます。
- プレイヤーの弾で全ての敵を倒すとクリアです。
- 敵の弾に当たるか敵が画面下部まで到達するとゲームオーバーになります。

## 操作方法

- `左右キー` : プレイヤー移動
- `Aボタン` : ショット
- `STARTボタン` : ゲーム開始 / 再開

## 実行方法

1. GBA エミュレータを用意します。
2. `インベーダーゲーム/game.bin` をエミュレータで読み込みます。

## ビルドについて

このプロジェクトは GBA 向けの直接メモリ操作を含んでいるため、GBA 用のクロスコンパイラ環境（例: devkitPro）が必要です。

標準的なビルド例:

```sh
arm-none-eabi-gcc -mthumb -march=armv4t -mthumb-interwork -nostdlib -T gba.ld -o game.elf "インベーダーゲーム/game コメントあり.c"
arm-none-eabi-objcopy -O binary game.elf "インベーダーゲーム/game.bin"
```

※ 上記コマンドは環境に応じて調整してください。

## 注意

- ソースは GBA のハードウェアに直接アクセスしています。
- PC 上の通常の C コンパイラでそのまま実行することはできません。
- エミュレータまたは実機での動作確認をおすすめします。
